# Rezsi Figyelő

Közüzemi mérőállás nyilvántartó webapp bérlők és ingatlan tulajdonosok számára.

## Funkciók
- Bérlői felület: PIN-es belépés, mérőállás rögzítés, fotó feltöltés, előzmények
- Admin felület: Dashboard, ingatlanok kezelése, tarifák, fizetések, karbantartás, ROI
- Mobil-first, PWA támogatás
- PostgreSQL adatbázis, Docker alapú deployment

## Tech Stack
- Flask (Python)
- PostgreSQL 15
- Chart.js (grafikonok)
- Docker + Docker Compose
- Gunicorn + Nginx (prod)
